 The IUCN Red List of Threatened Species Download Service Darwin Core Archive Export

This page provides links to zipped files of different versions of the IUCN Red List data in Darwin Core Archive (DwC-A) format. Each zipped file contains four text files (taxon, vernacularname, distribution, description) which include the taxonomy of all taxa published in that edition of the Red List, common names, Red List status and distribution data, and the narrative text fields supporting each assessment. The zipped file also includes an archive descriptor file (meta.xml) and an additional file describing the dataset as a whole (the eml.xml file).
